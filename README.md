# 3D Dungeon Crawler

A procedural 3D dungeon crawler built with **Blazor WebAssembly** and **Babylon.js**.

## 🛡️ Features
- **Procedural Dungeon**: Randomized room-and-corridor generator (no ceilings for better camera view).
- **Voxel Actors**: Warrior Vanguard, Swamp Goblins, and Greater Orcs rendered from JSON voxel definitions.
- **Weapon System**: Procedural Swords, Shields, Axes, and Maces.
- **Dynamic AI**: NPCs roam the dungeon with random pathing and idle states.
- **Smooth Animations**: Sine-wave based joint rotations for realistic walking.

## 🚀 Getting Started
1.  Clone the repository.
2.  Open in **Visual Studio 2022** or use **dotnet CLI**.
3.  Run the project (`dotnet run` or F5).
